# Library Management System

A simple Library Management System built with Python using Object-Oriented Programming (OOP).

## Features

- Add Book
- Display Books
- Search Book
- Add User
- Display Users
- Borrow Book
- Return Book
- Display Borrowed Books

## Technologies

- Python
- Object-Oriented Programming (OOP)

## Project Structure

```
Python-OOP-Projects/ Library_Management_System
│
├── main.py
│
├── models/
│   ├── book.py
│   └── user.py
│
└── managers/
    ├── admin.py
    └── operations_manager.py
```

## How to Run

1. Clone the repository.
2. Open the project in VS Code.
3. Run `main.py`.
