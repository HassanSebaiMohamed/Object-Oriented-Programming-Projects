# 🏥 Hospital Management System

A simple Hospital Management System built using **Python** and **Object-Oriented Programming (OOP)**.

## Features

- Add new patient
- Print all patients
- Get next patient
- Remove a leaving patient
- Handle urgent patients with higher priority

## Project Structure

```
Hospital-System/
│
├── main.py
├── patient.py
├── specialization.py
├── operations_manager.py
├── utility.py
└── README.md
```

## How to Run

```bash
python main.py
```

## OOP Concepts Used

- Classes & Objects
- Encapsulation
- Composition
- Dictionaries
- Lists
- Methods
- Input Validation
